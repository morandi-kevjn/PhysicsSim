# PhysicsSim

A simple 2D physics simulation in C#.

## ⭐️ Features

- Simulates multiple particles affected by gravity.
- Bouncing off walls.
- DOckerized for easy deployment.

## 🏢 Requirements

- Net SDK 8.0+
- Docker (optional, for containerized runs)

## 🚀 How to Run

### Locally

dotnet run

### Docker
docker build -t physics-sim .
docker run physics-sim
